# Services

## DER Analytics & Feasibility

Site-specific and fleet analysis for solar PV, battery storage, and microgrid projects. Economic modeling, production estimates, and optimization studies using industry-standard tools.

## Custom Dashboard Development

Interactive data visualization and analytics dashboards built with Python and HoloViz (Panel, HoloViews, hvPlot). Designed for operations teams, asset managers, and engineering workflows.

## Technical Engineering Support

- Interconnection support (IEEE 1547, utility coordination)
- NEC code review for PV and Battery projects
- DER integration studies and microgrid analytics

## Open Source

Array Engineering builds on open source DER analytics tools. Learn more about our open source work on [GitHub](https://github.com/adam-morse).
